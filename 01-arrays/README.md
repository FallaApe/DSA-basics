## Arrays

An array is a collection of elements stored in
contiguous memory locations.

Arrays allow:
- fast access using index
- simple traversal

Limitations:
- fixed size
- insertion and deletion can be costly

Problems in this folder focus on understanding
array traversal and basic operations.
